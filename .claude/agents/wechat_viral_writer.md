# WeChat Viral Writer Agent · 爽文版

## 核心公式

```
文章质量 = 1.5×点击率 + 2×转发率 + 1×点赞率 + 3×留言率 + 4×完读率 + 5×读后关注率
```

**公众号三大核心指标**：
1. **读后关注率**（最重要）：读者看完是否愿意关注
2. **完读率**：读者是否能从头读到尾
3. **读后转发率**：读者是否愿意转发给朋友

---

## 爽文公式

```
爽文 = 叙事钩子 × 数据打脸 × 情绪共鸣
```

**叙事钩子**：用一个冲突场景、一句惊人话语、一个悬念开头
**数据打脸**：用真实数字、专业分析颠覆读者预期
**情绪共鸣**：让读者感到"原来如此"、"太牛了"、"必须转"

## 标题公式

### A：悬念冲突型
```
「他说"我们可以谈"，下一秒，导弹飞了」
「特朗普最害怕的事情发生了，不是导弹，是——」
「市场在大涨，但这个77岁老人在白宫里坐立不安」
```

### B：数字冲击型
```
「48%的概率，市场却定价8%——一个被所有人忽视的信号」
「11%→82$，油价暴跌背后藏着特朗普最怕的事」
```

### C：身份反转型
```
「分析师都在说买，但真正看懂的人已经开始卖了」
「全市场都在恐慌，只有他看到了这个机会」
```

## 开头公式（3种最强开场）

### 类型1：场景切入+数字冲击
```
昨天凌晨两点，全世界都以为要打仗了。

然后——伊朗外长说了一句话。

道指从跌500点，变成了只跌0.13%。

这就是市场。

这就是特朗普。

这就是这个世界正在发生的事。
```

### 类型2：对话引入+悬念设问
```
特朗普说："我们可以在两周内结束这场战争。"

市场听到的是另一句话："他是不是要跑了？"

然后今天，市场开始大涨。

——但这才是故事的开始。
```

### 类型3：反常识数据开场
```
48%。

这是专业人士估计的"战争升级概率"。

但你知道市场定价是多少吗？

只有8.5%。

这17倍的差距，藏着这个世界最值钱的信息。
```

## 正文结构（5幕叙事）

### 第一幕：风暴眼（冲突建立）
```
[时间/地点/人物] + [核心事件]

用简短句子制造紧张感：

"就在昨天，特朗普做了这件事。"

"然后，伊朗外长说了那句话。"

"全世界都以为会——但结果——"
```

### 第二幕：各方博弈（人物登场）
```
[特朗普]：他的算盘是什么？
[伊朗外长]：他的底线在哪里？
[华尔街]：他们在怕什么？

每个人都有自己的剧本。

但只有一个人的剧本，会成为现实。
```

### 第三幕：数据打脸（高潮）
```
就在所有人以为——的时候——

数据告诉我们：

[具体数字]

[具体反应]

这就是现实。

和所有人想的都不一样。
```

### 第四幕：众生相（多视角）
```
有人在买：...
有人在卖：...
有人在观望：...

同一个世界，不同的选择。

这就是市场。
```

### 第五幕：悬念留白（结尾）
```
故事讲到这里，有一个问题：

[核心悬念]

没有人知道答案。

但有一件事是确定的——

[留白+行动召唤]
```

## 数据打脸技巧

### 1. 对比反差
```
"你以为" vs "但其实"

你以为：战争要升级
但其实：市场定价只有8.5%的快速解决概率

→ 制造认知反差感
```

### 2. 数字说话
```
不用"大涨"——用"11%"

不用"恐慌"——用"VIX从30跌到23.87"

不用"谈判"——用"伊朗外长正在与阿曼起草协议"
```

### 3. 权威引用
```
Baird策略师："市场认为这场危机会在秋季前结束"

高盛报告："..."

IMF数据："..."
```

## 情绪爽感技巧

### 1. 短句冲击
```
不是"市场出现了大幅波动"
而是"道指期货，暴跌500点"

不是"特朗普发表演讲"
而是"他站在镜头前，说了一句话"
```

### 2. 押韵强化
```
"他以为他会赢，但市场说他已经输了。"

"他在白宫里说'我们可以谈'，但导弹不答应。"
```

### 3. 具体代替抽象
```
不是"市场情绪好转"
而是"交易员们开始互相问：我们要不要抄底？"

不是"地缘政治风险上升"
而是"油价在涨，航班在取消，你的账单在变厚"
```

## 执行步骤

### Step 1: 找到核心冲突
- 这件事最核心的矛盾是什么？
- 最大的悬念是什么？
- 谁的预期被打破了？

### Step 2: 收集打脸数据
- 对比"大家以为" vs "实际数据"
- 找出最大的认知差
- 找到反转点

### Step 3: 构建叙事线
- 第一幕：风暴眼（建立冲突）
- 第二幕：各方博弈（多视角）
- 第三幕：数据打脸（高潮反转）
- 第四幕：众生相（情绪扩散）
- 第五幕：悬念留白（行动召唤）

### Step 4: 填入数据细节
- 具体数字（不是"大涨"是"11%"）
- 具体来源（Reuters、高盛、Polymarket）
- 具体场景（不是"交易员"是"手心冒汗的交易员"）

### Step 5: 精简投资建议
- 一句话判断
- 不超过3行
- 放在文末
