# Report to Viral · 公众号爆款生成器

将专业投研报告转换为高可读性、强传播性的公众号文章。

---

## 核心公式

```
爆款 = 情绪Hook × 干货密度 × 故事张力 × 视觉节奏
```

## 文章结构（5模块）

### 模块1：核心判断（结论先行）

```
格式：
## 一句话判断

[置信度可视化] ★★★☆☆ 可靠程度

[核心判断，1-2句话，结论先行]
```

### 模块2：演讲解码（可视化表格）

```
格式：
## 演讲解码

[升级语言 vs 降级语言对比表格]

[关键语句解读，3-5条，每条一行]
```

### 模块3：市场剧本（情景化）

```
格式：
## 市场剧本

### 剧本A：[情景名称]（概率%）
[一句话描述] → [市场反应]

### 剧本B：[情景名称]（概率%）
[一句话描述] → [市场反应]

### 剧本C：[情景名称]（概率%）
[一句话描述] → [市场反应]
```

### 模块4：仓位建议（可执行）

```
格式：
## 现在怎么操作？

| 资产 | 操作 | 仓位 | 入场价 | 止损位 |
|------|------|------|--------|--------|
| ... | ... | ... | ... | ... |

[每条建议一行，附入场价和止损位]
```

### 模块5：监测信号（行动导向）

```
格式：
## 接下来看什么？

[24-48小时关键信号列表，每条20字以内]
- 正面信号：...
- 负面信号：...
- 紧急监测：...
```

## 标题公式（3选1）

### 公式A：数字悬念型
```
[概率数字] + [核心事件]
「48%！中东最坏剧本正在逼近」
```

### 公式B：情绪反差型
```
[表面现象] vs [隐藏真相]
「特朗普说"快结束了"，但市场在疯抢黄金」
```

### 公式C：身份权威型
```
[权威身份] + [震撼事实]
「六位顶级分析师一致警告：这一仗躲不过去」
```

---

## 开头 Hook 公式（3步走）

```
Step 1: 场景切入（30-50字）
「凌晨两点，中东传来消息——伊朗向美国盟友发射了导弹」

Step 2: 核心数字锚定（20-30字）
「随后 2-3 周，特朗普发出最后通牒，全球市场瞬间暴涨 10%」

Step 3: 核心问题抛出（15-20字）
「这个仗，到底打还是不打？」
```

## 情绪节奏

| 位置 | 情绪 | 技巧 |
|------|------|------|
| 标题 | 悬念/焦虑 | 数字+感叹号 |
| 开头 | 紧张/期待 | 场景+问题 |
| 模块1 | 权威/确定 | 结论先行 |
| 模块2 | 好奇/解谜 | 表格对比 |
| 模块3 | 期待/紧张 | 情景分支 |
| 模块4 | 行动/紧迫 | 具体数字 |
| 模块5 | 期待/参与 | 互动引导 |
| 结尾 | 温暖/互动 | 转发+关注 |

## 执行步骤

### Step 1: 读取源报告
```python
# 读取 JSON 和 MD 文件
# 提取核心字段：
# - speech_id / 报告ID
# - confidence_score / 置信度
# - scenario_probabilities / 情景概率
# - final_trades / 交易建议
# - market_data / 市场数据
# - key_contradictions / 核心矛盾
# - monitoring_triggers / 监测信号
```

### Step 2: 生成标题候选
根据核心判断，从3个公式中选择最合适的1个，生成3个标题候选。

### Step 3: 生成开头 Hook
按「3步走」模板生成开头段落。

### Step 4: 生成5大模块
按顺序生成每个模块，自动执行术语转化。

### Step 5: 生成结尾
```
---

[分隔线]

[互动问题] 你觉得特朗普这次会退缩吗？评论区见👇

[转发引导] 转给需要了解中东局势的朋友

[来源] 分析来源：TACO 6-Agent 系统 | 置信度 68%
```

## 格式规范

| 规范 | 要求 |
|------|------|
| emoji 使用 | ≤5个/模块，总计≤20个 |
| 表格 | 优先于段落呈现数据 |
| 数字 | 阿拉伯数字优先（48%而非四成八） |
| 术语 | 全部转化为通俗表达 |
| 段落 | 每段≤4行，避免长段落 |
| 标题 | 加粗，无emoji |
